# Youtube Shorts Blocker

YouTube Shortsの視聴抑制と，指定サイトの利用時間制限を行うChrome拡張機能である．

## 目的

YouTube Shortsを開いたときに警告を表示し，Shortsを開いている間は5分おきに再度警告を表示する．

また，指定したサイトを一定時間利用した場合に，そのサイトを15分間ブロックする．

## 実装済み機能

- YouTube Shortsのページを開いたときに警告を表示する．
- YouTube Shortsを開いている間，5分おきに警告を表示する．
- YouTube内の画面遷移でShortsに移動した場合も警告を表示する．
- Shorts以外のYouTubeページでは警告を表示しない．
- オプション画面から対象サイトと上限時間を登録する．
- 表示中のタブが対象サイトの場合だけ，利用時間を1秒単位で計測する．
- 上限時間に達したサイトを15分間ブロックする．
- 今日の計測時間とブロック状態をオプション画面で確認する．
- 今日の計測時間とブロック状態をリセットする．

## 使い方

1. Chromeの拡張機能画面でデベロッパーモードを有効にする．
2. 「パッケージ化されていない拡張機能を読み込む」から `src` フォルダを選択する．
3. 拡張機能の詳細画面から「拡張機能のオプション」を開く．
4. 対象サイトに `example.com` のようなドメインを入力する．
5. 上限時間，分を入力して追加する．

## 実装方針

Chrome拡張機能のcontent scriptをYouTubeページ上で実行し，現在のURLがShortsページかどうかを判定する．

Shortsページかどうかは，URLのパスが `/shorts/` から始まるかで判定する．

YouTubeはページ全体を再読み込みせずに画面遷移するため，初回読み込み時だけでなく，URL変更も監視する．

Shortsページに入ったときは即時に警告を表示し，Shortsページに滞在している間だけ5分間隔のタイマーを動かす．Shortsページから離れた場合は，タイマーを停止する．

指定サイトの時間制限は，content scriptがページの可視状態を確認し，可視状態のときだけbackground service workerへ通知する．background service workerは通知間隔に基づいて対象サイトの利用時間を保存する．

利用時間は日付ごとに `chrome.storage.local` へ保存する．上限に到達した場合は，対象サイトのブロック終了時刻を保存し，content scriptが全画面オーバーレイを表示する．

## 開発方針

最初は `window.alert` を使って警告を表示する．

ただし，`window.alert` はブラウザ操作を一時停止するため，将来的にはページ内に独自の警告UIを表示する方式も検討する．

指定サイトのブロックUIは，ページ内に全画面オーバーレイを表示する方式で実装する．

## 注意点

content scriptの対象URLをShortsページだけに限定すると，YouTube内の通常ページからShortsへ移動した場合に検知できない可能性がある．そのため，content scriptはYouTube全体を対象にし，JavaScript側でShortsページかどうかを判定する設計を基本方針とする．

時間制限機能は，Chromeの内部ページや拡張機能ページでは動作しない．また，ブラウザが完全に閉じられている間は計測しない．
