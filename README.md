# Youtube Shorts Blocker

YouTube Shortsと指定サイトの利用時間を，猫の全画面オーバーレイで制限するChrome拡張機能である．

## 目的

YouTube Shortsや指定サイトを一定時間利用した場合に，猫の全画面オーバーレイを表示して休憩を強制する．

## 実装済み機能

- YouTube Shortsを標準ルールとして登録する．
- YouTube Shortsを5分利用した場合，猫の全画面オーバーレイで休憩を強制する．
- YouTube内の画面遷移でShortsに移動した場合も計測する．
- Shorts以外のYouTubeページは，YouTube Shorts標準ルールでは計測しない．
- オプション画面から対象サイト，上限時間，休憩時間を登録する．
- 登録済みルールの上限時間と休憩時間を変更する．
- 表示中のタブが対象サイトの場合だけ，利用時間を1秒単位で計測する．
- 上限時間に達したサイトを，登録した休憩時間だけブロックする．
- ブロック中は巨大な猫の全画面オーバーレイを表示する．
- 今日の計測時間とブロック状態をオプション画面で確認する．
- 今日の計測時間とブロック状態をリセットする．

## 使い方

1. Chromeの拡張機能画面でデベロッパーモードを有効にする．
2. 「パッケージ化されていない拡張機能を読み込む」から `src` フォルダを選択する．
3. 拡張機能の詳細画面から「拡張機能のオプション」を開く．
4. YouTube Shorts標準ルールの有効，無効や時間を確認，変更する．
5. 追加したい対象サイトに `example.com` のようなドメインを入力する．
6. 上限時間，分と休憩時間，分を入力して追加する．

## 実装方針

Chrome拡張機能のcontent scriptをページ上で実行し，現在のURLが登録ルールに一致するかどうかを判定する．

YouTube Shortsは標準ルールとして扱い，ホストが `youtube.com` またはそのサブドメインで，URLのパスが `/shorts/` から始まる場合だけ一致する．

YouTubeはページ全体を再読み込みせずに画面遷移するため，初回読み込み時だけでなく，現在URLに対する定期的な状態確認でShortsへの移動を検知する．

登録ルールに一致するページが表示中の場合だけ利用時間を加算する．対象ページから離れた場合は，そのページの利用時間は加算しない．

指定サイトの時間制限は，content scriptがページの可視状態を確認し，可視状態のときだけbackground service workerへ通知する．background service workerは通知間隔に基づいて対象サイトの利用時間を保存する．

利用時間は日付ごとに `chrome.storage.local` へ保存する．上限に到達した場合は，対象サイトのブロック終了時刻を保存し，content scriptが猫の全画面オーバーレイを表示する．

## 開発方針

ブロックUIは，ページ内に猫の全画面オーバーレイを表示する方式で実装する．

## 注意点

content scriptの対象URLをShortsページだけに限定すると，YouTube内の通常ページからShortsへ移動した場合に検知できない可能性がある．そのため，content scriptはYouTube全体を対象にし，JavaScript側でShortsページかどうかを判定する設計を基本方針とする．

時間制限機能は，Chromeの内部ページや拡張機能ページでは動作しない．また，ブラウザが完全に閉じられている間は計測しない．
